About Me

I'm Mustapha Sufiyan Yahaya, a passionate and skilled blind front-end developer with a strong focus on building accessible and inclusive web applications. I have a deep understanding of web accessibility standards (WCAG) and utilize assistive technologies like screen readers and braille displays in my development process. I'm proficient in using React, JavaScript, and CSS to create dynamic and interactive user interfaces that are both engaging and accessible to users with disabilities. Beyond technical skills, I possess strong problem-solving abilities and a keen eye for detail. I'm a collaborative team player with excellent communication skills, and I'm always eager to learn new technologies and best practices in web development. I'm dedicated to creating a more inclusive digital world where everyone has equal access to information and experiences on the web.
